- Standardization，常用 Z-score 标准化
- 计算公式
  collapsed:: true
	- $$
	  x' = \frac{x - \mu}{\sigma}
	  $$
	- `x'  = (当前值 - 该列平均值) / 该列的标准差`
	- `方差 = 每个值 和 均值的差 的平方和 的 平均值.`
	- `标准差计算公式: 方差开平方根`
- API调用
  collapsed:: true
	- ```python
	  # 2. 创建标准化对象.
	  transfer = StandardScaler()
	  # 3. 对原数据集进行标准化操作.
	  x_train_new = transfer.fit_transform(x_train)
	  ```
