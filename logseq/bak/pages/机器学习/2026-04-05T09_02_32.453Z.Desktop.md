		- 归一化
			- 计算公式
			  collapsed:: true
				- $$
				  x' = \frac{x - x_{\min}}{x_{\max} - x_{\min}}
				  $$
				- `x' = (当前值 - 该列最小值) / (该列最大值 - 该列最小值)`
			- API调用
			  collapsed:: true
				- ```python
				  # 创建归一化对象
				  transfer = MinMaxScaler()
				  # 对原数据集进行归一化操作
				  x_train_new = transfer.fit_transform(x_train)
				  ```
		- 标准化
			- 计算公式
			  collapsed:: true
				- $$
				  x' = \frac{x - \mu}{\sigma}
				  $$
				- `x'  = (当前值 - 该列平均值) / 该列的标准差`
				- `方差 = 每个值 和 均值的差 的平方和 的 平均值.`
				- `标准差计算公式: 方差开平方根`
			- API调用
			  collapsed:: true
				- ```python

	- 4种距离算法如何选择？
	  collapsed:: true
		- | 场景 | 推荐距离 | 原因 |
		  | ---- | ---- | ---- |
		  | 物理空间中的坐标（如房产位置） | **欧氏距离** | 直观反映直线距离 |
		  | 用户评分、高维稀疏向量 | **曼哈顿距离** | 对异常值不敏感，且高维中更稳定 |
		  | 特征重要性不均，只关心最差情况 | **切比雪夫距离** | 关注最大偏差 |
		  | 不确定该用哪个，需要调参 | **闵可夫斯基距离** | 通过调整 p*p* 值自动寻找最优 |
