	- ## 🧠 一句话说清楚（费曼）
		- ```text
		  Faithfulness（事实一致性）：召回 → 回答
		  Context Precision（上下文精确率）：问题 → 召回
		  Context Recall（上下文召回率）：召回 → 回答（覆盖率）
		  Answer Relevancy（答案相关性）：问题 → 回答
		  ```
		- 为方便说明以上概念，作者举例说明，为考察学生的学习能力，现在学校组织了一场开卷考试，开始前学校给学生发了一份参考资料（知识库），并告诉学生答案都在参考资料中，职责划分如下：
		  id:: 69c14d65-9a96-4c3b-aedf-4ff80ca178a0
			- 老师/用户（RAG系统的使用者），负责出题。
			  id:: 69c14d85-fd3b-4dc8-85f6-88b17bd03f4a
			- 学生负责从参考资料中查阅资料，并根据资料进行并作答。
			- 阅卷老师（GPT4-o等，一般要比学生更聪明）来给学生的表现打分
		- 现在老师/用户出了一道题如：`请说出Python中==、is和__eq__()的区别是什么？`
			- 检索阶段：接着学生拿着问题去查阅资料，结果他查到5（Top_k）个相关知识片段，
			- 生成阶段：然后根据这5个知识片段组织答案，然后将组织好的答案给阅卷老师
			- 评估阶段：阅卷老师根据正确答案对这位学生的表现打分。
		- 学生提交试卷后，阅卷老师从如下几个维度给学生分：
		  id:: 69c150e5-ab61-470b-8247-73679b06fc04
			- 1、查找到的资料对不对，考察的学生的检索能力
			  id:: 69c15114-a54b-47e0-b7e9-f2ff7ce575b5
				- 召回率：别漏掉答案 → 正确片段 / 所有正确片段的比率，代表着`召回` 和`回答`的关系
				  id:: 69c15479-3b91-4b19-b390-8094130766ed
				- 准确率：别检索垃圾 → 正确片段 / 召回的片段总数
				- 召回率和准确率的区别，举个例子🌰说明：有一个钱袋子放了100张人民币，其中有4张100元，某人捞出来5（Top_k）张，只捞到3张100，那么
					- 召回率（你找全了吗？）：3/4 = 0.75
					- 精确率（你找对了几个？）：3/5 = 0.6
				- 召回率和准确率的区别，举个例子🌰说明：有一个钱袋子放了100张人民币，其中有6张100元，某人捞出来5（Top_k）张，捞到5张100，那么
					- 召回率（你找全了吗？）：5/64= 0.83
					- 精确率（你找对了几个？）：355 = 1
				- 结论🏁：**精确率高，召回率不高，那么这时候就应该调大top k了**
				  id:: 72ceed20-d6a4-4e46-aa07-0edaeab2dd16
		-
		- 企业最佳实践：RAGAS + [[PAGAS]] 融合方案，如下：
			- ```python
			  # RAGAS 评估返回的 scores 示例（0-1 分制）
			  scores = {
			      "faithfulness": 0.95,        # RAGAS 事实一致性分数
			      "context_precision": 0.88,    # RAGAS 检索精准度分数
			      "context_recall": 0.82,       # RAGAS 上下文召回率分数
			      "answer_relevancy": 0.90,     # RAGAS 答案相关性分数
			      "ragas_score": 0.8875         # RAGAS 内置的默认平均分（可选）
			  }
			  
			  # 企业自定义的业务权重（PAGAS 思路）
			  business_weights = {
			      "faithfulness": 0.4,
			      "context_precision": 0.3,
			      "context_recall": 0.2,
			      "answer_relevancy": 0.1
			  }
			  
			  # 计算业务总分（PAGAS 总分）
			  total_score = sum(scores[key] * business_weights[key] for key in business_weights)
			  # 计算过程：0.95*0.4 + 0.88*0.3 + 0.82*0.2 + 0.90*0.1 = 0.906
			  ```
