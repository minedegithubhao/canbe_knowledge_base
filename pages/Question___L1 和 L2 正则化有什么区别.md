- **题目**
	- L1 和 L2 正则化有什么区别？

- **标准回答**
	- L1 正则化惩罚参数绝对值之和，倾向把部分参数压到 0，因此可以产生稀疏模型，也常用于特征选择。L2 正则化惩罚参数平方和，倾向让参数整体变小、更平滑，不会像 L1 那样强烈制造稀疏。两者都用于控制模型复杂度、缓解过拟合，但 L1 更偏稀疏和筛特征，L2 更偏稳定和平滑。

- **回答结构**
	- L1 惩罚绝对值之和。
	- L1 倾向稀疏，可做特征选择。
	- L2 惩罚平方和。
	- L2 倾向参数平滑缩小。
	- 都要用验证集调正则强度。

- **项目追问**
	- 正则化越强越好吗？
		- 不是。太强会导致欠拟合，需要用验证集或交叉验证选择强度。
	- 什么场景选 L1？
		- 特征很多、希望模型稀疏或需要做特征选择时。
	- 什么场景选 L2？
		- 希望模型稳定、保留大部分特征贡献、减少参数过大时。

	- 不要只说 L1 是一阶、L2 是二阶，要说稀疏和平滑的差异。
	- 要补一句正则强度过大也会欠拟合。
