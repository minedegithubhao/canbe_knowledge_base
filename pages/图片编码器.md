- 图片编码器（Multimodal Image Encoder）:负责将图片（像素）转换为与文本在同一向量空间的向量（也就是说维度数相同），从而实现图文对齐与跨模态理解（方便图片向量和文本向量相互计算）。
- # 主流实现方式（按企业使用率排序）
	- ViT（Vision Transformer）+ 文本编码器（最主流）
	- CNN + Transformer（兼顾速度与精度）
	- 专用多模态编码器（端侧 / 轻量）
- # 核心作用（企业真实场景）
	- 图文检索（RAG 多模态）
	- 图文对齐与理解
	- 跨模态特征提取
- # 与 Transformer 架构的关系
	- 多模态图片编码器本质是  [[Only-Encoder]] （ViT 就是 Transformer Encoder）。
	- 在多模态大模型中：
		- 图片编码器（ViT）→ 提取图片表示
		- 文本编码器（Transformer Encoder）→ 提取文本表示
		- 两者在同一向量空间对齐 → 实现跨模态能力。
- # 企业选型要点
	- **优先用 [[图文对比预训练模型（CLIP）]] 系列**：成熟、通用、效果好，适合绝大多数图文场景。
	- **算力有限**：用轻量版（如 CLIP-B/32、MobileViT）。
	- **领域专用**：在 CLIP 基础上做微调（如医疗、电商图片）。
