- **一句话定义**
	- 激活函数是在神经网络层之间加入非线性变换的函数，让多层网络能够表达复杂关系。

- **解决的问题**
	- 如果只有线性层，多层线性变换仍然等价于一个线性变换。
	- 现实任务中的输入和输出关系往往是非线性的。
	- 激活函数让网络能够学习复杂决策边界和表示。

- **常见类型**
	- Sigmoid
		- 输出在 0 到 1 之间，早期常用，但容易梯度消失。
	- Tanh
		- 输出在 -1 到 1 之间，比 Sigmoid 居中，但深层网络仍可能梯度消失。
	- ReLU
		- 小于 0 输出 0，大于 0 近似线性，计算简单，缓解梯度消失，是常见默认选择。
	- Softmax
		- 把一组 logits 转成概率分布，常用于多分类输出层，详见 [[Concept/Softmax]]。

- **项目中怎么用**
	- 隐藏层常从 ReLU 或其变体开始。
	- 输出层按任务选择：二分类常用 Sigmoid，多分类常用 Softmax，回归任务常用线性输出。
	- 激活函数会影响梯度传播、训练速度和模型表达能力。

- **面试表达**
	- 为什么神经网络需要激活函数？ #card
		- 因为没有激活函数时，多层线性层叠加仍然只是线性模型，无法表达复杂非线性关系。激活函数引入非线性，让网络能拟合复杂模式。

- **常见误区**
	- 误区：激活函数只是把数值压到固定范围。
		- 更核心的作用是引入非线性。
	- 误区：所有层都要用 Softmax。
		- Softmax 通常用于多分类输出或注意力权重归一化，不是普通隐藏层默认激活。
