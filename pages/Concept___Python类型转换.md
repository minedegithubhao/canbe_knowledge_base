- **一句话定义**
	- Python 类型转换是把一个对象转换成另一种类型的过程，例如把字符串转成整数、把列表转成元组、把整数转成字符串。

- **解决的问题**
	- 接口、文件和用户输入通常是字符串，需要转成业务需要的数值、列表或布尔判断。
	- 不同容器适合不同场景，需要在 list、tuple、set、dict 等结构之间转换。

- **核心机制**
	- `int`、`float`、`str` 用于常见数值和字符串转换。
	- `list`、`tuple`、`set` 可在可迭代对象之间转换。
	- `ord` 把字符转成整数编码，`chr` 把整数编码转成字符。
	- `eval` 会执行字符串里的表达式，存在安全风险，不应随意处理外部输入。

- **典型流程**
	- 明确原始数据类型。
	- 判断目标业务类型。
	- 使用显式转换函数。
	- 捕获转换失败的异常，避免脏数据直接中断流程。

- **项目中怎么用**
	- 读取配置、CSV、表单参数、环境变量时，经常需要把字符串转成 int、float、bool 或列表。
	- 数据清洗中要对异常值、空字符串和格式错误做兜底。

- **面试表达**
	- Python 类型转换要注意什么？ #card
		- 要显式处理非法输入和精度问题，避免直接对外部字符串使用 eval；金额和高精度小数不要直接依赖 float。

- **常见误区**
	- 误区：`eval` 是通用转换工具。
		- eval 会执行代码，处理不可信输入有安全风险。
	- 误区：所有字符串都能直接转 int。
		- 空字符串、带单位、带逗号或非法字符都会失败，需要清洗或异常处理。
