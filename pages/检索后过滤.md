	- ## 🧠 一句话说清楚（费曼）
		- 引入检索后过滤的核心目的：**止损幻觉**—— 当检索结果无效时，直接拒答，避免模型 “无中生有”；
		- 核心价值：大幅提升 Groundedness（忠实度），解决高风险场景下的幻觉问题，同时规避业务合规风险；
		- 落地关键：以 Top1 BGE 分数为判断依据，阈值建议通用场景 0.3、高风险场景 0.4，既不轻易拒答（影响用户体验），也不纵容幻觉。
		- | 解决的问题 | 具体影响 | 对应 PAGAS 指标优化 |
		  | ---- | ---- | ---- |
		  | 1. 根治 “无上下文幻觉” | 避免模型基于原生记忆编造答案，确保回答要么有检索依据，要么直接拒答 | Groundedness（忠实度）大幅提升（从 < 0.7→≥0.9） |
		  | 2. 提升答案可信度 | 对 “知识库外的问题” 明确拒答，而非给错误答案，符合金融 / 医疗等场景的合规要求 | Answer Relevance（相关性）≥0.85 |
		  | 3. 降低人工纠错成本 | 减少 “看似正确、实则错误” 的回答，降低业务侧人工校验的工作量 | 问题解决率（有效回答占比）提升 |
		  | 4. 规避业务风险 | 防止错误回答带来的合规风险（比如医疗 / 金融场景的误导性信息） | 高风险场景下的合规性达标 |
