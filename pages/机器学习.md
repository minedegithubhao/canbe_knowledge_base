		- 归一化
		  collapsed:: true
			- 计算公式
			  collapsed:: true
				- $$
				  x' = \frac{x - x_{\min}}{x_{\max} - x_{\min}}
				  $$
				- `x' = (当前值 - 该列最小值) / (该列最大值 - 该列最小值)`
			- API调用
			  collapsed:: true
				- ```python
				  # 创建归一化对象
				  transfer = MinMaxScaler()
				  # 对原数据集进行归一化操作
				  x_train_new = transfer.fit_transform(x_train)
				  ```
		- 标准化
		  collapsed:: true
			- 计算公式
			  collapsed:: true
				- $$
				  x' = \frac{x - \mu}{\sigma}
				  $$
				- `x'  = (当前值 - 该列平均值) / 该列的标准差`
				- `方差 = 每个值 和 均值的差 的平方和 的 平均值.`
				- `标准差计算公式: 方差开平方根`
			- API调用
			  collapsed:: true
				- ```python
