	- 特征预处理
	  collapsed:: true
		- | 方法名称 | 核心作用 | 典型使用场景 |
		  | ---- | ---- | ---- |
		  | [[归一化]] | 将特征值**缩放到 [0,1] 或指定区间**，消除不同特征的量纲和数值大小差异，让所有特征处于同一尺度，避免大数值特征主导模型训练 | 1. 基于距离计算的模型（如 KNN、SVM、K-Means 聚类）2. 神经网络输入层（激活函数多在 0-1 区间）3. 数据分布无明显异常、需要保留原始分布结构的场景4. 图像处理（像素值归一化到 0-1） |
		  | [[标准化]] | 将特征值转换为**均值为 0、标准差为 1**的标准正态分布，消除量纲影响，同时保留数据的离散程度信息，增强模型对异常值的鲁棒性 | 1. 基于梯度下降的线性模型（如线性回归、逻辑回归）2.  PCA 降维、主成分分析（要求特征服从正态分布）3. 数据存在异常值、需要抗干扰的场景4. 树模型以外的大多数机器学习模型（如 SVM、神经网络） |

		- | 算法名称 | 核心作用 | 典型使用场景 |
		  | ---- | ---- | ---- |
		  | [[交叉验证]] | 把数据集划分成多折，轮流作为训练 / 验证集，**验证模型评估结果的稳定性与可靠性**，防止过拟合 | 模型训练完成后，评估泛化能力、对比不同模型性能、作为模型选择的依据 |

		  | [[梯度下降]]  | 迭代计算损失函数的梯度方向，**不断更新模型参数**，使得损失值最小化，训练模型 | 模型训练阶段，最小化代价函数，更新权重 / 偏置，用于收敛模型 |

		- 距离度量算法
		  collapsed:: true
			- | 算法名称 | 核心作用 | 对异常值敏感 | 典型使用场景 |
			  | ---- | ---- | ---- |
			  | [[欧氏距离]] | 两个点各维度差值的平方和开平方 | 高 | 1. 低维、连续数值型数据（如身高、体重）2. KNN、K-Means 等基础聚类 / 分类算法3. 特征已做归一化 / 标准化、无明显异常值的场景 |
			  | [[曼哈顿距离]] | 两个点各维度差值的绝对值之和（城市街区距离） | 中 | 1. 高维稀疏数据、特征维度差异大的场景2. 网格状路径规划（如城市导航）3. 特征量纲一致、需要抗轻微异常值的场景 |
			  | [[切比雪夫距离]] | 两个点各维度差值的绝对值的最大值（也叫棋盘距离) | 极高（只看最大差） | 1. 物流配送、棋盘路径规划等场景2. 只需要关注「最大差异维度」的场景3. 特征维度重要性差异极大的场景 |
			  | [[闵可夫斯基距离]] | 欧氏、曼哈顿、切比雪夫距离的**通用公式**，通过参数 p 统一三种距离 | 随 p 值变化p=1 → 曼哈顿距离；p=2 → 欧氏距离；p→∞ → 切比雪夫距离； | 1. 需要灵活切换距离度量的场景2. 算法调参时，通过调整 p 值优化模型效果3. 统一管理多种距离度量的通用场景 |

		- | 指标名称 | 公式描述（纯中文） | 核心作用 | 使用场景 |
		  | ---- | ---- | ---- |
		  | [[均方误差]]  | 所有预测误差的平方取平均值 | 衡量预测误差的整体大小，对大误差有极强的惩罚性，是回归模型最常用的损失函数 | 平方放大误差，对 outliers 零容忍，适合模型训练 |
		  | [[平均绝对误差]] | 所有预测误差的绝对值取平均值 | 衡量预测值与真实值的平均偏差，对异常值鲁棒性强，线性惩罚误差 | 绝对值线性惩罚，抗异常值能力最强，适合脏数据场景 |
		  | [[均方根误差]] | 对均方误差（MSE）开平方 | 保留 MSE 对大误差的惩罚性，同时单位与原数据一致，直观反映平均误差水平 | MSE 的业务友好版，单位和原数据一致，汇报首选 |
