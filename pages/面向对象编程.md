	- 魔术方法
	  collapsed:: true
		- `__init__()` 方法：初始化方法或构造方法
		  collapsed:: true
			- ```python
			  # 1、定义一个类
			  class Person():
			      # 初始化实例对象属性
			      def __init__(self, name, age):
			          # 赋予name属性、age属性给实例化对象本身
			          # self.实例化对象属性 = 参数
			          self.name = name
			          self.age = age
			  
			  # 2、实例化对象并传入初始化属性值
			  p1 = Person('孙悟空', 500)
			  # 3、调用p1对象自身属性name与age
			  print(p1.name)
			  print(p1.age)
			  ```
		- `__str__()` 方法：print对象时调用，默认打印对象的内存地址，可以通过该方法修改print的输出内容
		  collapsed:: true
			- ```python
			  # 1、定义一个类
			  class Car():
			      # 首先定义一个__init__方法，用于初始化实例对象属性
			      def __init__(self, brand, model, color):
			          self.brand = brand
			          self.model = model
			          self.color = color
			  
			      # 定义一个__str__内置魔术方法，用于输出小汽车的相关信息
			      def __str__(self):
			          return f'汽车品牌：{self.brand}，汽车型号：{self.model}，汽车颜色：{self.color}'
			  
			  # 2、实例化对象c1
			  c1 = Car('奔驰', 'S600', '黑色')
			  print(c1)
			  ```

		- ⚠️私有属性不能在类的外部被直接访问，会报错
	- **私有属性**的设置和修改：`get_xx `用来获取私有属性，`set_xx `用来修改私有属性值。
	  collapsed:: true
		- ```python
		  class Girl():
		      def __init__(self, name):
		          self.name = name
		          self.__age = 18
		  
		      # 公共方法：提供给外部的访问接口
		      def get_age(self):
		          # 内部访问：允许直接访问
		          # 外部访问：根据需求要添加限制条件
		          return self.__age
		  
		      # 公共方法：提供给外部的设置接口
		      def set_age(self, age):
		          self.__age = age
		  
		  
		  girl = Girl('小美')
		  girl.set_age(19)
		  print(girl.get_age())
		  ```

	- 代码实现
	  collapsed:: true
		- ```python
		  '''
		  首先定义一个父类，其可能拥有多个子类对象。当我们调用一个公共方法（接口）时，传递的对象不同，则返回的结果不同。
		  '''
		  class Fruit(object):
		      def makejuice(self):
		          print('i can make juice')
		  
		  class Apple(Fruit):
		      # 重写父类方法
		      def makejuice(self):
		          print('i can make apple juice')
		  
		  class Banana(Fruit):
		      # 重写父类方法
		      def makejuice(self):
		          print('i can make banana juice')
		  
		  class Orange(Fruit):
		      # 重写父类方法
		      def makejuice(self):
		          print('i can make orange juice')
		  
		  # 定义一个公共接口（专门用于实现榨汁操作）
		  def service(obj):
		      # obj要求是一个实例化对象，可以传入苹果对象/香蕉对象
		      obj.makejuice()
		  
		  # 调用公共方法
		  service(Orange())
		  ```
- 类属性
  collapsed:: true
	- 类属性属于类；子类可以访问，但是不能修改
	- ```python
	  class Person(object):
	      # 定义类属性count
	      count = 0
	  
	      def __init__(self, name):
	          self.name = name
	          Person.count += 1
	  
	  # 1、实例化对象p1
	  p1 = Person('Tom')
	  p2 = Person('Harry')
	  p3 = Person('Jennifer')
	  
	  print(f'通过实例访问类属性：{p1.count}')  # 优先访问实例属性，找不到则访问类属性
	  print(f'通过类名访问属性：{Person.count}')
	  
	  p1.count = 10  # 通过实例不能修改类属性
	  print(f'通过实例访问类属性：{p1.count}')  # 有了实例属性，就访问实例属性
	  print(f'通过类名访问属性：{Person.count}')
	  Person.count = 100
	  print(f'通过类名修改类属性：{p2.count}')
	  ```
- 类方法
  collapsed:: true
	- 类方法 → 修改类属性，`classmethod`
	- ```python
	  class Person(object):
	      # 定义类属性count
	      count = 0
	  
	      def __init__(self, name):
	          self.name = name
	          Person.count += 1
	  
	      @classmethod
	      def get_count(cls):
	          print(f'我们共实例化了{cls.count}个对象')
	  
	  
	  # 1、实例化对象p1
	  p1 = Person('Tom')
	  p2 = Person('Harry')
	  p3 = Person('Jennifer')
	  
	  Person.get_count()  # 通过类方法访问类属性
	  ```
- 静态方法
  collapsed:: true
	- ① 既 不需要访问实例属性或者调用实例方法
	- ② 也 不需要访问类属性或者调用类方法
	- 使用关键字`staticmethod`
	- ```python
	  # 开发一款游戏
	  class Game(object):
	      # 开始游戏，打印游戏功能菜单
	      @staticmethod
	      def menu():
	          print('1、开始游戏')
	          print('2、游戏暂停')
	          print('3、退出游戏')
	  
	  
	  # 开始游戏、打印菜单
	  Game.menu()
	  ```
