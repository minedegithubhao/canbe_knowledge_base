- 字符串切片
	- ```python
	  # 序列名称[开始位置下标:结束位置下标:步长(步阶)]
	  三个参数都有默认值: 
	      开始位置默认值: 0
	      结束位置默认值: -1 (最后一个)
	      步长 :  1
	  # 步长可以为负数，正数代表从左向右截取，负数代表从右向左截取
	  numstr = '0123456789'
	  numstr[0:3:1]  # 012 => range方法非常类似，步长：每次前进1步
	  
	  ```
