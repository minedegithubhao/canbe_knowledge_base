- **一句话定义**
	- 多模态 LLM 是能处理文本、图片、音频、视频等多种模态输入，并进行理解或生成的大模型。

- **解决的问题**
	- 让模型不只理解文本，还能结合视觉、语音或其他模态信息完成任务。

- **核心机制**
	- 不同模态先经过对应编码器。
	- 编码结果被对齐或投影到 LLM 可处理的表示空间。
	- LLM 基于多模态上下文进行理解和生成。

- **典型流程**
	- 图片、文本或音频输入。
	- 模态编码器提取特征。
	- 投影或对齐到语言模型空间。
	- LLM 生成回答。

- **项目中怎么用**
	- 图文问答、图片理解、截图分析、视觉 Agent 都依赖多模态 LLM。

- **面试表达**
	- 多模态 LLM 和多模态 RAG 有什么区别？ #card
		- 多模态 LLM 是模型能力，能直接理解多模态输入；多模态 RAG 是检索增强架构，会先检索文本、图片等证据，再让模型回答。

- **常见误区**
	- 误区：多模态 LLM 等于图文 RAG。
		- 一个是模型能力，一个是应用架构。
