	- ## 🧠 一句话说清楚（费曼）
		- **嵌入表**本质是一个**二维数组**（矩阵），**行数 = 词表大小**，**列数 = 向量维度**
		- 可以实现Token ID到 [[初始向量]]的检索，[[嵌入]]就是根据Token ID查询初始[[向量]]的过程
		- | Token ID | 向量维度 1 | 向量维度 2 | 向量维度 3 | 向量维度 4 |
		  | ---- | ---- | ---- |
		  | 0（我） | 0.12 | -0.34 | 0.56 | 0.78 |
		  | 1（今天） | 0.90 | -0.11 | 0.23 | 0.45 |
		  | 2（吃饭） | 0.67 | 0.89 | -0.01 | 0.32 |
