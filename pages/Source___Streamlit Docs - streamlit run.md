- **原始信息**
	- Streamlit 官方 `streamlit run` 命令文档。

- **核心观点**
	- `streamlit run` 用于启动 Streamlit 应用。
	- 可以传入入口文件或目录；不传入时，Streamlit 会尝试运行当前目录下的 `streamlit_app.py`。
	- 命令可以附带配置选项和脚本参数。
