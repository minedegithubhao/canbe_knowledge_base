- **题目**
	- Python 面向对象三大特性是什么？

- **标准回答**
	- 面向对象三大特性是封装、继承和多态。封装把对象状态和行为组织在一起，通过公开方法控制外部访问；继承让子类复用和扩展父类公共能力；多态让不同对象通过同一调用方式表现出不同行为。
	- Python 中多态不一定强依赖继承，更常体现为鸭子类型：调用方关注对象是否具备需要的行为，而不是对象具体属于哪个类。

- **回答结构**
	- 先列三大特性。
	- 分别解释封装、继承、多态。
	- 补充 Python 的鸭子类型特点。
	- 结合项目说类如何降低复杂状态管理成本。

- **项目追问**
	- Python 的私有属性是真私有吗？
		- 不是绝对私有。双下划线会触发名称改写，主要用于减少外部误用，不是安全边界。
	- 什么时候不用类？
		- 如果逻辑无状态、参数简单、复用边界清晰，普通函数通常更直接。
