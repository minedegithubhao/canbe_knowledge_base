- **一句话定义**
	- Python 函数是可调用的一等对象，用来封装可复用逻辑，也可以被传递、返回和装饰。

- **解决的问题**
	- 把重复逻辑封装成可维护单元。
	- 通过参数传入变化点，通过返回值输出结果。
	- 为闭包、装饰器、回调和函数式写法提供基础。

- **核心机制**
	- 函数可以返回一个值；多个返回值通常由 tuple 承接。
	- 参数常见形式包括位置参数、关键字参数、默认参数、不定长位置参数和不定长关键字参数。
	- 一个星号用于展开序列参数，两个星号用于展开字典参数。
	- global 用于在函数内重新绑定全局变量，nonlocal 用于在内层函数中重新绑定外层函数变量。
	- lambda 适合写短小匿名函数，常作为排序 key 或简单回调，不适合承载复杂逻辑。

- **典型流程**
	- 先定义清晰的输入和输出。
	- 默认参数放在非默认参数之后。
	- 可变默认参数改用空值占位，再在函数体内创建新对象。
	- 函数内部尽量减少对外部可变状态的隐式修改。

- **项目中怎么用**
	- 数据清洗中用函数封装字段转换、过滤规则和聚合逻辑。
	- 排序或分组时常用 lambda 作为短小 key 函数。
	- `isinstance`、`print`、`input` 等内置函数常用于调试、输入输出和类型判断。
	- 工程代码中，函数边界要表达清楚是否修改传入对象、可能抛什么异常。

- **面试表达**
	- Python 函数参数有哪些类型？ #card
		- 常见有位置参数、关键字参数、默认参数、不定长位置参数和不定长关键字参数。还要能说明默认参数顺序、可变默认参数坑，以及用一个星号和两个星号做参数解包。

- **常见误区**
	- 误区：多个返回值是真正返回多个独立对象。
		- Python 函数本质返回一个对象，多个返回值通常被打包成 tuple。
	- 误区：lambda 更高级，应该尽量多用。
		- lambda 适合短表达式；复杂逻辑应写普通函数，便于命名、调试和测试。
