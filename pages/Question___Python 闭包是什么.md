- **题目**
	- Python 闭包是什么？

- **标准回答**
	- 闭包是内部函数引用外部函数作用域变量，并且这个内部函数被返回或传递后，仍然能访问这些外部变量的结构。形成闭包通常需要三个条件：嵌套函数、内部函数引用外部变量、外部函数返回内部函数。
	- 闭包可以保存少量状态，常用于函数工厂、回调和装饰器。内层函数如果要重新绑定外层变量，需要使用 nonlocal。

- **回答结构**
	- 先给闭包定义。
	- 说清楚三个形成条件。
	- 解释外层变量为什么还能被访问。
	- 补充用途和 nonlocal。

- **项目追问**
	- 闭包和类怎么选？
		- 状态少、行为简单时闭包轻量；状态复杂、行为多、需要清晰生命周期时类更可读。
	- 闭包和装饰器是什么关系？
		- 装饰器通常用闭包实现，包装函数引用被装饰的原函数。

- **我的易错点**
	- 不要只说“函数里套函数”，要说内部函数引用外部变量并被继续使用。
	- 不要忘记修改外层局部变量需要 nonlocal。
