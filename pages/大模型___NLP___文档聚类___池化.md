	- ## 🧠 一句话说清楚（费曼）
		- 池化（Pooling）就是：把一段文本里的多个词向量，变成 1 个向量。
		- 整篇文档 → 模型输出一堆词向量 → 池化 → 1 个文档向量 → 聚类
	- ## 💘企业开发场景
		- [[平均池化]]（80% 场景）：稳定、通用，适合大多数文档聚类 / 检索；
			- ((69afb869-65fa-4481-8791-f24ced4d0ab3))
		- [[CLS 池化]]（15% 场景）：需要灵活调节池化软硬程度时用（比如个性化推荐、精准检索）
			- ((69afb7e4-3303-4d43-8911-4b8d9c577a18))
		- [[Max 池化]]（5% 场景）：只在需要突出关键特征时用（比如关键词匹配、异常检测）。
			- ((69afba34-e4fc-48be-991c-897b023e16eb))
		- id:: 69afb8ad-7217-4ca1-a8f3-b42263d45dd4
		  ```text
		  import torch
		  
		  # 模拟输入：3个chunk，每个chunk是4维向量（[N_chunks, C] = [3,4]）
		  logits = torch.tensor([
		      [1.0, 2.0, 3.0, 4.0],
		      [2.0, 3.0, 4.0, 5.0],
		      [3.0, 4.0, 5.0, 6.0]
		  ])
		  
		  # 1. mean池化（平均）
		  mean_pool = pool_chunk_logits(logits, mode="mean")
		  # 输出：[[2.0, 3.0, 4.0, 5.0]] → 每个维度的平均值
		  
		  # 2. max池化（最大）
		  max_pool = pool_chunk_logits(logits, mode="max")
		  # 输出：[[3.0, 4.0, 5.0, 6.0]] → 每个维度的最大值
		  
		  # 3. lse池化（tau=1.0）
		  lse_pool = pool_chunk_logits(logits, mode="lse", tau=1.0)
		  # 输出：[[2.4076, 3.4076, 4.4076, 5.4076]] → 介于mean和max之间
		  ```
