	- ## 🧠 一句话说清楚（费曼）
		- 这是通过“强行转换”让两边数据站在同一起跑线上的方法。
		- **做法**：把两组得分都缩放到 $0$ 到 $1$ 之间。
			- **公式**：$Score_{new} = \frac{Score - Min}{Max - Min}$
			- **例子**：如果关键词最高分是 $20$，那 $20$ 就变成 $1$，$10$ 就变成 $0.5$。
		- **融合**：归一化后，你就可以按比例加权了。
			- $Total = 0.3 \times 关键词归一化分 + 0.7 \times 向量归一化分$
		- **缺点**：非常不稳定。如果某次搜索出现了一个分值极高的“怪胎”（离群值），会把其他所有文档的分数都压得很低，导致排序失准。

		  | **特性** | [[归一化权重]] | [[RRF]] (倒数排名融合) |
		  | ---- | ---- | ---- |
		  | **参考依据** | 原始分数值 | 排名名次 |
		  | **计算复杂度** | 略高（需要算全局 Max/Min） | 极低 |
		  | **调优难度** | 高（需要手动分配权重比例） | 低（几乎不需要调参） |
		  | **稳定性** | 容易受极端分数干扰 | 非常稳定 |
