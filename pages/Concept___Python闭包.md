- **一句话定义**
	- Python 闭包是内部函数引用外部函数作用域变量，并在外部函数返回后仍能访问这些变量的结构。

- **解决的问题**
	- 在不使用类的情况下保存少量状态。
	- 为装饰器、回调、延迟执行和函数工厂提供基础。
	- 将状态封装在函数作用域中，减少全局变量。

- **核心机制**
	- 形成闭包通常需要嵌套函数、内部函数引用外部变量、外部函数返回内部函数。
	- 外部函数执行结束后，被内部函数引用的变量不会立即释放。
	- 内部函数只读取外层变量时不需要 nonlocal。
	- 内部函数要重新绑定外层变量时，需要使用 nonlocal。

- **典型流程**
	- 外层函数创建需要保存的状态。
	- 内层函数读取或更新这些状态。
	- 外层函数返回内层函数。
	- 调用返回的函数时，它仍能访问外层状态。

- **项目中怎么用**
	- 简单计数器、配置化函数、回调函数和装饰器都可以用闭包实现。
	- 如果状态复杂、生命周期长或行为很多，类通常比闭包更清晰。

- **面试表达**
	- Python 闭包是什么？ #card
		- 闭包是内部函数引用外部函数变量，并且内部函数被返回后仍能访问这些变量。答题要说出三个要点：嵌套函数、引用外部变量、返回内部函数。

- **常见误区**
	- 误区：闭包就是内部函数。
		- 只有内部函数引用了外部作用域变量，并被外部继续使用时，才体现闭包价值。
	- 误区：修改外层变量直接赋值就行。
		- 内层函数要重新绑定外层局部变量，需要 nonlocal。
