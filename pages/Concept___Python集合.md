- **一句话定义**
	- Python 集合是基于哈希的无重复元素容器，适合去重、成员判断和集合运算。

- **解决的问题**
	- 快速判断某个元素是否存在。
	- 对数据去重。
	- 做交集、并集、差集等集合运算。

- **核心机制**
	- set 中元素不重复。
	- set 不保证按插入顺序表达业务语义。
	- set 元素必须可哈希。
	- 空集合不能用空花括号创建，空花括号是空 dict。

- **典型流程**
	- 去重时把可哈希元素放入 set。
	- 做成员判断时使用 in。
	- 添加元素用 add。
	- 删除元素时区分 remove 和 discard 的异常行为。

- **项目中怎么用**
	- 用 set 保存已经处理过的 id，避免重复处理。
	- 用 set 做黑白名单、权限集合或标签集合的快速判断。
	- 用集合运算比较两批数据差异。

- **面试表达**
	- set 和 list 在成员判断上有什么差异？ #card
		- list 通常需要线性扫描，set 基于哈希表，平均成员判断更快。但 set 要求元素可哈希，并且不用于表达稳定顺序。

- **常见误区**
	- 误区：set 是有序容器。
		- set 不应依赖顺序表达业务逻辑。
	- 误区：空集合写成空花括号。
		- 空花括号创建的是 dict，空 set 要显式创建。
