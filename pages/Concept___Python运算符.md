- **一句话定义**
	- Python 运算符是表达计算、比较、逻辑判断、成员判断和对象身份判断的语法工具。

- **解决的问题**
	- 表达数值计算、条件分支、循环判断、集合包含关系和对象引用关系。

- **核心机制**
	- 算术运算符包括加、减、乘、除、整除、取模和幂运算。
	- 比较运算符用于生成布尔结果。
	- 逻辑运算符包括 `and`、`or`、`not`，具有短路特性。
	- 成员运算符 `in`、`not in` 用于判断元素是否存在。
	- 身份运算符 `is`、`is not` 判断两个名字是否指向同一个对象。

- **典型流程**
	- 数值计算先明确除法和整除语义。
	- 条件判断先明确比较对象和布尔短路。
	- 判断相等用 `==`，判断是否同一对象用 `is`。
	- 判断容器成员用 `in`。

- **项目中怎么用**
	- 数据清洗、特征过滤、接口参数校验和业务规则判断都会大量使用运算符。
	- `is None` 是判断空值哨兵的常见写法，不能用来判断普通值相等。

- **面试表达**
	- `==` 和 `is` 有什么区别？ #card
		- `==` 比较值是否相等，`is` 比较是否是同一个对象。判断 None 通常用 `is None`，判断普通值相等用 `==`。

- **常见误区**
	- 误区：`/` 和 `//` 一样。
		- `/` 是普通除法，`//` 是整除。
	- 误区：`is` 可以替代 `==`。
		- `is` 判断对象身份，不判断值相等。
