- 查
	- ```python
	  userTuple = ('张三', '李四', '王五', 10, 20, 30, True, False)
	  # 使用索引查找元素
	  元组[索引]
	  userTuple[1] # 输出： 李四
	  # 查找元素
	  userTuple.index('王五') # 输出： 2
	  # 统计数量
	  userTuple.count('王五') # 输出： 1
	  # 元组长度
	  len(userTuple) # 输出：8
	  ```
