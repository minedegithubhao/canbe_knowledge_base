- **一句话定义**
	- 维度数是向量中数字的个数，也可以理解为模型表示对象时使用的特征空间大小。

- **解决的问题**
	- 描述向量、嵌入表、隐藏状态等张量形状。
	- 帮助理解模型容量和表示空间。

- **核心机制**
	- 一个三维向量有 3 个数字。
	- 一个 768 维 token 向量有 768 个数字。
	- 嵌入表列数通常就是 embedding 维度。

- **典型流程**
	- Token ID 查嵌入表。
	- 得到固定维度的 token 向量。
	- 进入 Transformer 后保持或变换维度。

- **项目中怎么用**
	- 选择 embedding 模型、向量库维度、模型隐藏层大小时都需要关心维度数。

- **面试表达**
	- 向量维度数是什么意思？ #card
		- 它表示向量里有多少个数字。模型用这些数字组成的空间表示 token、文本或图片的特征。

- **常见误区**
	- 误区：维度越高一定越好。
		- 维度更高可能表达能力更强，但也会增加存储、计算和检索成本。
