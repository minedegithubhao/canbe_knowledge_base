- **一句话定义**
	- 张量是深度学习中承载数据的多维数组，可以表示标量、向量、矩阵和更高维数据。

- **解决的问题**
	- 为模型训练和推理提供统一的数据结构。
	- 支持 GPU 加速、自动求导和批量计算。

- **核心机制**
	- 0 维张量是标量。
	- 1 维张量是向量。
	- 2 维张量是矩阵。
	- 更高维张量可以表示 batch、序列、通道、图像等结构。

- **典型流程**
	- 分词器输出 Token ID 列表。
	- 转成张量。
	- 输入模型。
	- 模型输出 logits、hidden states 或 embedding 张量。

- **项目中怎么用**
	- PyTorch 中模型输入、权重、激活和输出都用张量表示。

- **面试表达**
	- 张量和向量有什么关系？ #card
		- 向量是一维张量，矩阵是二维张量。张量是更泛化的多维数组，是深度学习框架中的核心数据结构。

- **常见误区**
	- 误区：张量只是一种数学概念。
		- 在 PyTorch 里张量也是实际的数据载体，包含 dtype、shape、device 等信息。
