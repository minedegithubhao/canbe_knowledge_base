- 匹配方式
	- match匹配
	  collapsed:: true
		- 只能从**开头开始**匹配,只能返回**第一个**匹配成功的内容
		- ```python
		  import re
		  
		  result = re.match('bcd', 'abcdefgbcd')
		  print(result)  # None
		  if result:
		      print('匹配成功', result.group())
		  else:
		      print('匹配失败!')
		  ```
	- search匹配
	  collapsed:: true
		- 可以**整个字符**串搜索,只能返回**第一个**匹配成功的内容
		- ```python
		  import re
		  
		  result = re.search('bcd', 'abcdefgbcd')
		  print(result)  # <re.Match object; span=(1, 4), match='bcd'>
		  if result:
		      print('匹配成功', result.group())  # bcd
		  else:
		      print('匹配失败!')
		  ```
	- findall匹配
		- 可以整个字符串搜索,可以返回**所有匹配成功**的内容
		- 匹配成功:返回匹配成功数据的列表->[元素1,元素2,元素3...]
		- 匹配失败: 返回空列表->[]
		- ```python
		  import re
		  
		  print('-------------------------')
		  result = re.findall('bcd', 'abcdefgbcd')
		  print(result) # ['bcd', 'bcd']
		  ```
- 匹配规则
	- 单个字符规则
		- | 符号 | 含义 |
		  | ---- | ---- | ---- |
		  | `.` | 任意 1 个字符（除换行符 `\n`） |
		  | `[]` | 匹配中括号内任意 1 个字符 |
		  | `\d` | 任意 1 个数字（0-9） |
		  | `\D` | 任意 1 个非数字字符 |
		  | `\s` | 任意 1 个空白字符（空格、`\t`、`\n` 等） |
		  | `\S` | 任意 1 个非空白字符 |
		  | `\w` | 任意 1 个单词字符（字母、数字、下划线、汉字） |
		  | `\W` | 任意 1 个非单词字符（特殊符号） |
		- `d`:digit数字  `s`:space空白   `w`:word词
	- 多个字符规则
	  collapsed:: true
		- | 符号 | 含义 | 匹配次数要求 |
		  | ---- | ---- | ---- |
		  | `*` | 匹配前一个字符出现 0 次或无限次 | 次数 ≥ 0 |
		  | `+` | 匹配前一个字符出现 1 次或无限次 | 次数 ≥ 1 |
		  | `?` | 匹配前一个字符出现 0 次或 1 次 | 次数 = 0 或 次数 = 1 |
		  | `{x}` | 匹配前一个字符出现 x 次 | 次数 = x |
		  | `{x,y}` | 匹配前一个字符出现 x 到 y 次 | x ≤ 次数 ≤ y |
		- `{x,y}`会匹配尽可能长，也就是y的长度
	- 开头和结尾
	  collapsed:: true
		- | 符号 | 含义 | 说明 |
		  | ---- | ---- | ---- |
		  | `^` | 匹配以指定内容**开头** | 用于限定字符串起始位置 |
		  | `$` | 匹配以指定内容**结尾** | 用于限定字符串结束位置 |
		- `^` 和 `$` 通常**配合使用**，用于精确限制整个字符串的格式。
	- 分组
	  collapsed:: true
		- | 符号 | 含义 | 说明 |
		  | ---- | ---- | ---- |
		  | 竖线 | 匹配左右任意一个表达式 | 表示 “或者” 逻辑 |
		  | `()` | 分组，默认编号从 1 开始 | 括号内内容为一个整体，自动生成数字编号 |
		  | `\n` | 根据分组编号引用分组 | `n` 为分组数字，引用第 n 个括号匹配的内容 |
		  | `(?P<name>)` | 为分组设置别名 | 给当前分组命名为 `name` |
		  | `(?P=name)` | 根据别名引用分组 | 引用别名为 `name` 的分组匹配内容 |
	- 特殊匹配
		- 注意: 如果^放到了[]里面,表示取反作用
		- 举例: `[^\d]` == `[\D]`
- 贪婪模式和非贪婪模式
  collapsed:: true
	- 贪婪模式: 在整个表达式匹配成功的前提下，尽可能多的匹配
	- 非贪婪模式: 在整个表达式匹配成功的前提下，尽可能少的匹配
	- 正则中的量词：*和+，默认都是贪婪模式的匹配，可以在他们后面加?将其变为非贪婪模式。
	- 常用贪婪模式如:`.*`    `.+`
	- 常用非贪婪模式如: `.*?`   `.+?`
	- ```python
	  # 导包
	  import re
	  
	  # 已知要匹配的字符串如下
	  my_str = '<h1>编程语言</h1>此处省略很多内容<h1>python大数据</h1>'
	  
	  # .* : 贪婪模式,匹配成功的前提下,尽可能多的匹配
	  data = re.findall('<h1>.*</h1>', my_str)
	  print(data)  # ['<h1>编程语言</h1>此处省略很多内容<h1>python大数据</h1>']
	  print(len(data))  # 1
	  
	  # .*?: 非贪婪模式,匹配成功的前提下,尽可能少的匹配
	  data = re.findall('<h1>.*?</h1>', my_str)
	  print(data)  # ['<h1>编程语言</h1>', '<h1>python大数据</h1>']
	  print(len(data))  # 2
	  
	  # 需求: 只要标签中的标题名
	  # 知识点:findall匹配中如果有分组,那么只把对应分组里的内容放到列表中返回
	  data = re.findall('<h1>(.*?)</h1>', my_str)
	  print(data)  # ['编程语言', 'python大数据']
	  print(len(data))  # 2
	  ```
	- 知识点:findall匹配中如果有分组,那么只把对应分组里的内容放到列表中返回
- 正则修饰符
  collapsed:: true
	- re.I : 设置当前匹配不区分大小写
	- re.S : 设置.真正匹配到任意字符包含`\n`
	- ```python
	  # 导包
	  import re
	  
	  # 演示不区分大小写的匹配:re.I
	  # 注意: 验证码必须绝对匹配,可以使用^和$配合起到限制作用
	  data = re.findall('^a8D9$', 'A8d9', re.I)
	  print(data)  # ['A8d9']
	  
	  # 演示.真正匹配到任意字符包含\n: re.S
	  print('------------------------')
	  my_str = '123\n456\n789'
	  print(my_str)
	  print('------------------------')
	  data = re.match('.{7}', my_str)
	  print(data)  # None  因为.不能匹配到\n
	  print('------------------------')
	  data = re.match('.{7}', my_str, re.S)
	  print(data.group())  # 123\n456  因为re.S让.匹配到了\n
	  ```
