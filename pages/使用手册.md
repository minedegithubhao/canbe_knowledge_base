# 语法
	- 隐藏语法： {{cloze 企业真实痛点：脏数据、重复、噪音、乱码，聚类全}} 是 {{cloze 噪音}}
	- 使用别名时，如果有`英文全称`、`中文`，那么都写上，例如：
	  > alias:: Natural Language Processing,自然语言处理
- # 规则
	- 背诵内容标记如下
		- `#card`,`#面试背诵汇总/XXX/XXX`
	- 参考内容标记如下：
		- `#XXX/XXX`
	- 对需要解释`是什么？`的概念一律创建页面，对于英文概念以`简称`作为文档标题
	- 对于概念类的页面，使用[[Templates/技术概念类]]，使用方法
		- 使用`/template` 导入`技术概念模板`
		- 优点
			- **tags + alias**：保证永远搜得到
			- **一句话定义**：适合理解记忆、可在面试开口
			- **企业开发场景：**了解企业实际开发中如何使用
		- 查询：`{{query (property :type "概念")}}`，页面[[概念类汇总]]
	- 建立错题集，记录面试/学习中的错误，使用 [[Templates/错题集]]，使用方法
		- 使用`/template` 导入`错题本`
		- 优点
			- 可检索：tags + type 能一键筛选所有错题
			- 可追踪：掌握程度让你不做假努力
			- 只抓重点：错因 + 正确答案 + 回扣知识点用
		- 查询：`{{query (property :type "错题")}}`，页面[[错题汇总]]
