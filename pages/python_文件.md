- open函数打开文件
	- ```python
	  # 返回的结果是一个file文件对象
	  # name：是要打开的目标文件名(绝对路径/相对路径)
	  # mode：设置打开文件的模式(访问模式)：只读r、写入w、追加a等。
	  f = open(name, mode)
	  ```
		- > r模式：只读取操作。文件不存在 = 直接报错，光标放在文件的第一行。
		- > w模式：只写模式，文件不存在 = 创建，光标置于第一行 + 清空原有所有文件内容。
		- > a模式：追加模式，文件不存在 = 创建，尾部追加内容。
- 入门案例
	- ```python
	  # 1- 打开文件
	  f = open(file="./data/a.txt",mode='w')
	  # 2- 执行相关操作
	  f.write("hijklmn")
	  # 3- 关闭文件
	  f.close()
	  ```
- 指定打开文件编码格式
	- ```python
	  f = open('python.txt', 'w', encoding='utf-8')
	  ```
- 文件的读取操作
	- ```python
	  # 一、读取所有内容
	  # 1、打开文件
	  f = open('python.txt', 'r', encoding='utf-8')
	  # 2、使用read()方法读取文件所有内容
	  contents = f.read()
	  print(contents)
	  # 3、关闭文件
	  f.close()
	  
	  # 二、读取制定长度，💡可读非文本类文件
	  # 2、读取1024个字符长度文件内容，字母或数字
	  contents = f.read(1024)  
	  
	  # 三、一次读取所有文本内容，按行放入列表
	  # 2、读取文件
	  lines = f.readlines()
	  for line in lines:
	      print(line, end='')
	      
	  # 四、逐行读取
	  while True:
	     # 读取一行内容
	     content = f .readline()
	     # 判断是否读取到内容
	     if not content:
	         break
	     # 如果读取到内容，则输出
	     print(content)
	  
	  
	  ```
	- | **模式** | **描述** |
	  | ---- | ---- | ---- |
	  | r | 以只读方式打开文件。文件的指针将会放在文件的开头。这是默认模式。 |
	  | rb | 以二进制格式打开一个文件用于只读。文件指针将会放在文件的开头。这是默认模式。 |
	  | r+ | 打开一个文件用于读写。文件指针将会放在文件的开头。 |
	  | rb+ | 以二进制格式打开一个文件用于读写。文件指针将会放在文件的开头。 |
	  | w | 打开一个文件只用于写入。如果该文件已存在则打开文件，并从开头开始编辑，即原有内容会被删除。如果该文件不存在，创建新文件。 |
	  | wb | 以二进制格式打开一个文件只用于写入。如果该文件已存在则打开文件，并从开头开始编辑，即原有内容会被删除。如果该文件不存在，创建新文件。 |
	  | w+ | 打开一个文件用于读写。如果该文件已存在则打开文件，并从开头开始编辑，即原有内容会被删除。如果该文件不存在，创建新文件。 |
	  | wb+ | 以二进制格式打开一个文件用于读写。如果该文件已存在则打开文件，并从开头开始编辑，即原有内容会被删除。如果该文件不存在，创建新文件。 |
	  | a | 打开一个文件用于追加。如果该文件已存在，文件指针将会放在文件的结尾。也就是说，新的内容将会被写入到已有内容之后。如果该文件不存在，创建新文件进行写入。 |
	  | ab | 以二进制格式打开一个文件用于追加。如果该文件已存在，文件指针将会放在文件的结尾。也就是说，新的内容将会被写入到已有内容之后。如果该文件不存在，创建新文件进行写入。 |
	  | a+ | 打开一个文件用于读写。如果该文件已存在，文件指针将会放在文件的结尾。文件打开时会是追加模式。如果该文件不存在，创建新文件用于读写。 |
	  | ab+ | 以二进制格式打开一个文件用于追加。如果该文件已存在，文件指针将会放在文件的结尾。如果该文件不存在，创建新文件用于读写。 |
