- **题目**
	- Python 函数参数有哪些类型？

- **标准回答**
	- Python 常见函数参数包括位置参数、关键字参数、默认参数、不定长位置参数和不定长关键字参数。调用时可以按位置传，也可以按参数名传；默认参数必须放在非默认参数后面。
	- 不定长位置参数会把多余位置参数收集成 tuple，不定长关键字参数会把多余关键字参数收集成 dict。调用函数时，一个星号可以展开序列，两个星号可以展开字典。
	- 工程上要特别注意默认参数不要直接使用可变对象。

- **回答结构**
	- 先列参数类型。
	- 再解释不定长参数收集成什么。
	- 再解释传参解包。
	- 最后补充默认参数顺序和可变默认参数坑。

- **项目追问**
	- 为什么默认参数不要用空列表？
		- 默认参数在函数定义时创建一次，多次调用会共享同一个列表，容易产生跨调用状态污染。
	- lambda 和普通函数怎么选？
		- lambda 适合短表达式和临时 key 函数；复杂逻辑用普通函数，便于命名、调试和测试。
