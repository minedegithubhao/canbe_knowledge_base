- **一句话定义**
	- Python 字符串是不可变的文本序列，支持查找、切片、替换、分割、格式化和编码转换。

- **解决的问题**
	- 表达文本、路径、标识符和序列化内容。
	- 对文本做查找、切片、清洗和拼接。
	- 在文件、网络和数据库边界处理编码。

- **核心机制**
	- str 是不可变对象，replace、split、title 等方法返回新对象或新结果，不会原地修改原字符串。
	- find 找不到时返回特殊位置值，index 找不到时会抛异常。
	- split 返回列表，适合把结构化文本拆成字段。
	- 字符串切片会根据起止位置和步长生成新字符串。
	- 字符串格式化常用百分号格式、`format` 和 f-string，其中 f-string 更适合现代 Python 日常使用。

- **典型流程**
	- 查找子串时根据失败行为选择 find 或 index。
	- 替换文本时接收 replace 返回的新字符串。
	- 分割文本时使用 split 并处理空字段、空格和异常格式。
	- 需要高频拼接大量字符串时，优先考虑 join。

- **项目中怎么用**
	- 日志解析、路径处理、CSV 轻量清洗和接口字段处理都会频繁使用字符串方法。
	- 涉及文件和网络时，要明确文本编码，避免隐式编码问题。

- **面试表达**
	- Python 字符串为什么说是不可变对象？ #card
		- 字符串方法不会原地改变原字符串，而是返回新字符串或其他结果。变量重新赋值只是把名字绑定到新对象，不是修改原对象。

- **常见误区**
	- 误区：replace 会修改原字符串。
		- replace 返回新字符串，不修改原字符串。
	- 误区：find 和 index 的失败行为一样。
		- find 找不到返回特殊位置值，index 找不到会抛异常。
