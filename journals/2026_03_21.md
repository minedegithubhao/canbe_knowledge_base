	- 真实性 (Faithfulness)—— 查“韩老魔”有没有瞎编
		- ```python
		  from ragas.metrics import faithfulness
		  from ragas.llms.prompt import Prompt
		  
		  faithfulness_prompt = Prompt(
		      name="faithfulness_zh",
		      instruction="""
		      你是一名资深的修仙小说考据专家。
		      任务：判断【大模型回答】中的每一个事实陈述，是否都能在【检索到的原始片段】中找到依据。
		      
		      判定准则：
		      1. 角色对齐：韩立、韩老魔、厉飞雨、二愣子、韩道友均视为同一人。
		      2. 逻辑关联：允许通过多个碎片段拼接得出结论。例如片段A提到“祭出青色小鼎”，片段B提到“虚天鼎威力巨大”，则支持“韩立使用虚天鼎”的陈述。
		      3. 拒绝脑补：如果片段中完全没提某件法宝或功法，即便模型回答对了（可能来自模型自带知识），也要判为 0（无证据）。
		      
		      返回格式：JSON 数组，包含每个事实的判断结果（1为有据，0为幻觉）。
		      """
		  )
		  faithfulness.long_form_answer_prompt = faithfulness_prompt
		  ```

		      """
